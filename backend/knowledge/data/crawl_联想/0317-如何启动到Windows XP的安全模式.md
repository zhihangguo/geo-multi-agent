# 知识库 317

## 标题
如何启动到Windows XP的安全模式

## 问题描述
Windows XP的安全模式的启动步骤。

## 分类
问题类别: 安全模式

## 关键词
启动, Windows XP, 安全模式

## 元信息
创建时间:2024-12-15|版本:2.0

## 解决方案
1、出现开机LOGO画面时连续点击“**F8**”键 ，如下图：  
  
（图示仅供举例，可能与您所看到的画面不同）；

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/23595fd908efadcaa4638d1d60d9d20d-1714295286298.png&name=mceclip2.png)

2、进入“**Windows高级选项菜单**”，用上下方向键选择“**安全模式**”启动选项 ，按“**回车**”键，如下图：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/44084f555da79d6dcfbf66ef10fb397c-1714295271065.png&name=mceclip1.png)

3、选择要启动的操作系统，按“**回车**”键，如下图：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/c14bea4fa7b6865d32baa4c2b630254d-1714295303490.png&name=mceclip3.png)

4、正在加载windows文件，

5、在弹出的“**桌面**”对话框，选择“**是**”，如下图：

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/fd32f9e84dcf2034884b6f1b85407439-1714295347223.png&name=mceclip5.png)

6、进入Windows XP安全模式。

![](https://chinakb.lenovo.com.cn/chinakb/prod-api/file/downloadFile?key=uniko/IMAGE/fd8c8e16998f48328af9c8cb8e1e24d6-1714295360084.png&name=mceclip6.png)
