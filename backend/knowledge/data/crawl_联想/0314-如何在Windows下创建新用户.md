# 知识库 314

## 标题
如何在Windows下创建新用户

## 问题描述
如果一台计算机有多名使用者，可以通过不同的用户来分开使用，下文将指导如何在Windows下创建新用户。

## 分类
主类别: 操作系统故障
子类别: 账户相关

## 关键词
显示屏, 管理, 设置, 显示, 硬件, Windows, 查看, 用户, 创建

## 元信息
创建时间:2024-12-15|版本:1.0

## 解决方案
Windows下创建新用户 Windows XP、Windows 7/8/8.1 创建方法类似，本文以Windows 7创建为例。

1、点击“计算机”鼠标右键“管理”，如下图：  
  
![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/175.files/image001.jpg)  
  
2、点击“本地用户和组”--->“用户”在右侧空白处，单击鼠标右键，创建“新用户”；  
  
![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/175.files/image002.jpg)  
  
3、请输入“用户名”、“密码”，勾选“用户不能更改密码”与“密码永不过期”，点击“创建”；  
  
![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/175.files/image003.jpg)  
  
4、用户创建完毕。  
  
![](http://robotrs.lenovo.com.cn/ZmptY2NtYW5hZ2Vy/p4data/Rdata/Rfiles/175.files/image004.jpg)
