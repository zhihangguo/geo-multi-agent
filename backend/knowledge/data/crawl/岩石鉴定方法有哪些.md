# 岩石鉴定方法有哪些（野外与室内综合）

岩石鉴定通常采用“野外快速判别 + 室内精细鉴定”的两阶段方法。野外强调效率与初筛，室内强调准确与定量。

## 一、野外快速鉴定方法

### 1. 肉眼观察法
重点观察以下要素：
1. 颜色：整体颜色、斑点颜色、风化面与新鲜面的差异。  
2. 结构构造：是否层状、片理状、块状、气孔状、杏仁状等。  
3. 矿物组合：可见矿物种类及含量（如石英、长石、云母、角闪石等）。  
4. 粒度与结晶程度：粗粒、中粒、细粒，隐晶质或玻璃质。  
5. 定向性特征：矿物是否有明显定向排列（常见于变质岩）。

### 2. 手标本物理性质测试
1. 硬度（简易）：用小刀、钢针、玻璃做划痕对比。  
2. 条痕与断口：观察粉末颜色、贝壳状断口或参差断口。  
3. 比重感：同体积标本“手掂轻重”进行经验判断。  
4. 磁性测试：用小磁铁判断含磁性矿物情况。  
5. 稀盐酸反应：滴加稀盐酸，判断是否含碳酸盐矿物（是否起泡）。

### 3. 地质背景约束法
结合产状和区域地质背景进行判别：
1. 与围岩关系（侵入接触、整合/不整合接触）。  
2. 所处构造环境（断裂带、褶皱核部、盆地边缘等）。  
3. 区域岩浆岩/沉积岩/变质岩分布规律。  
4. 共生矿物组合与蚀变特征。

## 二、室内精细鉴定方法

### 1. 薄片镜下鉴定（偏光显微镜）
用于确定矿物种类、含量、结构和成因特征，是岩石学常规方法之一。

### 2. XRD（X射线衍射）
